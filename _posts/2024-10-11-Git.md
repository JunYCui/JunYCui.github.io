## Git Character
版本控制系统（Version System System, VCS）是一类用于追踪源代码（或其他文件、文件夹）改动的工具。\
Git 拥有一个经过精心设计的VCS模型，这使其能够支持版本控制所需的所有特性，例如维护历史记录、支持分支和促进协作。
